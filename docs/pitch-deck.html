How We Built It

Agentic AI ile Hızlandırılmış Geliştirme

Cursor IDE ekosistemi; mimari kısıtları anlayan, KVKK kurallarını uygulayan ve kodu üreten bir geliştirme ortağı olarak çalıştı.

📐

Cursor Ruleset (AGENTS.md)

Her modül için bağlam farkındalıklı kural seti. Agent, Clean Architecture'ı otomatik koruyarak kod üretti.

# RoadScore Ruleset
- Handlers → /internal/handlers
- HF timeout: 10s hard limit
- Rate limit middleware: zorunlu
- Error: fmt.Errorf("%w")
⚡

Cursor CLI & SDK

Handler iskelet üretimi, test senaryoları ve API entegrasyonu CLI komutlarıyla otomatize edildi. Ekstra puan kriterleri eksiksiz karşılandı.

🤖 Agent Döngüsü — 5 Adım

1
Mimari Prompt → Agent tüm Clean Architecture layer'larını (handler/usecase/repo) tek seferde üretti
2
Compose Mode → HF API + retry logic + timeout pipeline'ı tek promptta çalışır hale getirdi
3
Codebase Context → KVKK kurallarını tüm dosyalara otomatik blur middleware olarak ekledi
4
Hata Döngüsü → Linter hatası → Agent kendi düzeltti, açıkladı, anlamlı commit önerdi
5
Paralel Dev → 3 modül eş zamanlı, ayrı agent context penceresiyle bağımsız geliştirildi
